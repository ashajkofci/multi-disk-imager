# bNovate Multi Disk Imager

<img src="src/MultiDiskImager.App/Assets/AppIcon.png" width="128" alt="bNovate Multi Disk Imager icon">

## Reliable disk deployment, without proprietary image formats

bNovate Multi Disk Imager gives technical teams a focused desktop tool for creating, deploying, and validating raw disk images across Windows, macOS, and Ubuntu. One image can be written to several removable devices in parallel, helping reduce repetitive deployment time while preserving a transparent, byte-for-byte image format.

### Built for controlled operations

- Write one image to multiple target devices concurrently.
- Read physical media into standard raw `.img` files.
- Verify deployed media and identify the first mismatching byte.
- Use MD5, SHA-1, or SHA-256 checksums for image validation.
- Limit reads to allocated MBR or GPT partitions when a compact image is needed.
- Review device model, capacity, and platform identifier before destructive actions.
- Keep the system disk excluded from selectable targets.
- Track progress, throughput, estimated time remaining, and per-device results.

### Ready for mixed workstation environments

Self-contained releases are produced for Windows x64, macOS on Intel and Apple Silicon, and Ubuntu on x64 and ARM64. The application automatically follows the system language and includes English, French, German, Italian, Spanish, Portuguese, Dutch, Polish, Simplified Chinese, and Japanese.

### Open images, predictable workflows

Images remain raw byte-for-byte data. There is no proprietary container, header, or metadata layer, so existing imaging, forensic, validation, and archival tools can continue to work with the same files.
